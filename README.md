# ReactKit
A react project starter.
Use react redux react-redux react-router react-hot-loader eslint and so on.

自己做的一个react项目的脚手架，包含：react redux react-redux react-router react-hot-loader eslint等。功能非常少，但是适合做集成用。

## Installation

```bash
npm install
```

## Running Dev Server

```bash
npm run dev
```

## Building Production

```bash
npm run build
```
